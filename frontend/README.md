# Frontend - Delivery Scheduling UI

Angular 18 single-page application for delivery scheduling.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start
# or
ng serve
```

App runs at: `http://localhost:4200`

## Build for Production

```bash
npm run build
```

Output: `dist/delivery-frontend/`

## API Configuration

Edit `src/environments/environment.ts` to change the backend API URL.
