# Backend - Delivery Scheduling API

Spring Boot REST API for delivery time slot scheduling.

## Quick Start

```bash
# Run tests
mvn test

# Start application
mvn spring-boot:run
```

Server runs at: `http://localhost:8080`

## H2 Console

For development debugging: `http://localhost:8080/h2-console`
- JDBC URL: `jdbc:h2:mem:deliverydb`
- Username: `sa`
- Password: (empty)
